# 📊 Market Dashboard - Frontend

## 🚀 Overview

This is the frontend application for the **Market Dashboard** project.
It provides a clean UI to search stocks, view historical data, and display market-related insights.

* 📊 Interactive UI for stock details

## 🧑‍💻 Local Setup

### 1. Clone the repo

```bash
git clone https://github.com/rishabh-147/market-dashboard-frontend/
cd <market-dashboard-frontend>
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure Backend URL

Update API base URL in your config (example):

```js
const BASE_URL = "https://market-dashboard-backend-h3i2.onrender.com";
```

### 4. Run locally

```bash
npm start
```
